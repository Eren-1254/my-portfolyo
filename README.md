# 🚀 Eren'in Yazılım Portfolyosu

Merhaba! Ben Afyon Kocatepe Üniversitesi Bilgisayar Mühendisliği 2. sınıf öğrencisiyim. Bu site, kendimi web teknolojileri ve yazılım alanında geliştirmek
için hazırladığım ilk projemdir.

## 🛠️ Kullanılan Teknolojiler
- **HTML5** (Yapısal düzen)
- **CSS3** (Modern tasarım ve Flexbox)
- **Responsive Design** (Mobil uyumluluk çalışmaları)

## 📁 Sergilenen Projeler
Sitemde şu an için iki ana C# projem yer almaktadır:
1. **Dice Game:** C# Windows Forms ile geliştirilen bir oyun.
2. **ATM Simulation:** C# Console tabanlı bankacılık mantığı çalışması.

## 🎯 Hedeflerim
Şu an aktif olarak JavaScript ve Unity (C#) üzerine çalışıyorum. Gelecekte bağımsız bir oyun geliştiricisi olmayı hedefliyorum.
